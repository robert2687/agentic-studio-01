# **App Name**: Agentic Studio

## Core Features:

- File System Explorer: Display a file system tree, showing the project's files and folders.
- Agent Status: Show the status of the agents in the agentic workflow, including progress, name, and any steps they have performed.
- Agent Orchestration: The orchestrator takes a plain text request, then converts that into steps for the agents.
- Chat View: Display chat messages between the user and orchestrator. Displaying what is happening with different Agents.
- Code Editor: View selected file code in the studio
- Gemini Canvas Integration: Allows Agents to generate canvas content.
- Live Preview: Simulated webapp rendering to allow the Agents to see how the webapp would look to a user.

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to convey trust and innovation.
- Background color: Dark gray (#1F2937) to provide a modern and focused environment.
- Accent color: Teal (#14B8A6) for highlighting key elements and interactive components.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, 'Inter' (sans-serif) for body text.
- Code font: 'Source Code Pro' (monospace) for code snippets and terminal outputs.
- Use modern, minimalist icons from Lucide-React for a clean and consistent look.
- Split the UI into logical panels: file explorer, code editor, chat, and preview, to facilitate efficient workflow.
